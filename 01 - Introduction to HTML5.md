# 01 - Introduction to HTML5

## Front-end Development

Front-end Development is the process of developing the user interface (UI) of a web application. It focuses on the visual part of the application that users can see and interact with.

Technologies used in Front-end Development:

* HTML5
* CSS3
* JavaScript

---

Full Stack Development is the process of developing both the front-end and back-end of an application.

### Layers of Full Stack Development

1. Front-end

   * HTML5

   * JavaScript

2. Back-end

   * Java
   * Python

   * Oracle SQL

## HTML

HTML stands for **Hyper Text Markup Language**.

### Hyper Text

Hyper Text means text that contains links to other pages or documents.

### Markup Language

Markup Language means a language that uses tags to structure and display content properly.

HTML does not perform calculations. It only defines the structure of a web page.

---

## History of HTML

1. In 1989, Tim Berners-Lee proposed the concept of HTML while working at CERN.

2. In 1991, the first version of HTML was released.

3. In 1993, HTML became publicly available and widely used.

4. On 22 January 2008, HTML5 was officially introduced.

## HTML Tags

HTML communicates with the browser using **Tags**.

Anything written inside angle brackets (`<>`) is called a Tag.

Examples:

```html
<h1>Hello World</h1>

<p>Welcome to HTML</p>
```

---

## Types of Tags

There are two types of HTML Tags:

### 1. Paired Tags

Paired Tags have both opening and closing tags.

Syntax:

```html
<tagname>
Content
</tagname>
```

Example:

```html
<h1>HTML Introduction</h1>
```

Here:

* `<h1>` is the opening tag
* `</h1>` is the closing tag

---

### 2. Unpaired Tags

Unpaired Tags do not require a closing tag.

Examples:

```html
<br>

<hr>

<img>
```

These tags perform specific operations and do not contain content.

---

## Angle Brackets

```html
<>
```

These symbols are called **Angle Brackets**.

Examples:

```html
<p>Paragraph</p>

<h1>Heading</h1>
```

* `<p>` → Starting Tag
* `</p>` → Ending Tag
