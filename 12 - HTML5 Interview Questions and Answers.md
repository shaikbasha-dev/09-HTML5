12 - HTML5 Interview Questions and Answers

1. What is HTML?

Answer:

HTML stands for Hyper Text Markup Language.

It is used to create and structure web pages.

2. What is HTML5?

Answer:

HTML5 is the latest version of HTML.

It provides:

Semantic tags
Audio and Video support
Form validation
Canvas
SVG
Local Storage
3. What is the difference between HTML and HTML5?

Answer:

HTML	HTML5
Older version	Latest version
No audio/video support	Supports audio and video
Limited form validation	Built-in validation
No local storage	Supports local storage
4. What is a Tag in HTML?

Answer:

A tag is used to communicate with the browser.

Example:

<p>Hello</p>

<p> is the opening tag.

</p> is the closing tag.

5. What are Paired Tags?

Answer:

Tags that have opening and closing tags.

Example:

<h1>Welcome</h1>
6. What are Unpaired Tags?

Answer:

Tags that do not require a closing tag.

Examples:

<br>
<hr>
<img>
<input>
7. What is <!DOCTYPE html>?

Answer:

It tells the browser that the document is an HTML5 document.

8. What is the purpose of <head>?

Answer:

The <head> tag contains:

Title
Meta tags
CSS links
JavaScript links
9. What is the purpose of <body>?

Answer:

The <body> tag contains the visible content displayed in the browser.

10. What are Heading Tags?

Answer:

HTML provides six heading tags:

<h1> to <h6>

<h1> is the largest.

<h6> is the smallest.

11. What is the difference between <b> and <strong>?

Answer:

<b> makes text bold.

<strong> makes text bold and indicates important text.

12. What is the difference between <i> and <em>?

Answer:

<i> displays italic text.

<em> displays emphasized text with semantic meaning.

13. What is Hyperlink?

Answer:

A hyperlink connects one webpage to another.

Created using:

<a href="">
14. What is the use of href?

Answer:

href specifies the destination URL.

Example:

<a href="https://www.google.com">
15. What is target="_blank"?

Answer:

It opens the link in a new browser tab.

16. What is the purpose of alt attribute?

Answer:

alt displays alternate text when an image cannot be loaded.

Example:

<img src="java.png" alt="Java Logo">
17. What is the difference between Ordered and Unordered Lists?

Answer:

Ordered List:

<ol>

Displays numbers.

Unordered List:

<ul>

Displays bullets.

18. What is a Description List?

Answer:

Description List uses:

<dl>
<dt>
<dd>

Used for terms and definitions.

19. Which tags are used to create a table?

Answer:

<table>
<tr>
<th>
<td>
20. What is <tr>?

Answer:

<tr> stands for Table Row.

21. What is <th>?

Answer:

<th> stands for Table Header.

22. What is <td>?

Answer:

<td> stands for Table Data.

23. What is colspan?

Answer:

It allows one cell to occupy multiple columns.

Example:

<td colspan="2">
24. What is rowspan?

Answer:

It allows one cell to occupy multiple rows.

Example:

<td rowspan="3">
25. What is a Form?

Answer:

A form collects user input.

Created using:

<form>
26. What is the difference between GET and POST?

Answer:

GET:

Data visible in URL
Less secure

POST:

Data hidden
More secure
27. What is the purpose of action attribute?

Answer:

It specifies where form data is sent.

Example:

<form action="save.java">
28. What is required attribute?

Answer:

Makes a field mandatory.

Example:

<input required>
29. What is placeholder?

Answer:

Displays hint text inside an input box.

Example:

<input placeholder="Enter Name">
30. What is pattern attribute?

Answer:

Used for validation using Regular Expressions.

Example:

<input pattern="[0-9]{10}">
31. What is id attribute?

Answer:

It uniquely identifies an HTML element.

32. What is class attribute?

Answer:

It groups multiple elements together.

Used in:

CSS
JavaScript
33. What is Semantic HTML?

Answer:

Semantic tags describe the meaning of content.

Examples:

<header>
<nav>
<section>
<article>
<footer>
34. What is the difference between <div> and <span>?

Answer:

<div>

Block-level element

<span>

Inline element
35. What is <audio> tag?

Answer:

Used to play audio files.

Example:

<audio controls>
36. What is <video> tag?

Answer:

Used to play videos.

Example:

<video controls>
37. What is controls attribute?

Answer:

Displays:

Play button
Pause button
Volume control
38. What is autoplay?

Answer:

Automatically starts audio or video.

39. What is poster attribute?

Answer:

Displays an image before a video starts.

Example:

<video poster="cover.jpg">
40. What are Global Attributes?

Answer:

Attributes that can be used with almost all HTML elements.

Examples:

id
class
style
title
hidden
lang
41. What is Local Storage in HTML5?

Answer:

Local Storage stores data in the browser permanently.

Methods:

localStorage.setItem()
localStorage.getItem()
42. What is Session Storage?

Answer:

Stores data temporarily.

Data is deleted when the browser tab is closed.

43. What is Canvas in HTML5?

Answer:

Canvas is used to draw graphics using JavaScript.

Example:

<canvas>
44. What is SVG?

Answer:

SVG stands for Scalable Vector Graphics.

It is used to create vector images.

45. What are the new features of HTML5?

Answer:

HTML5 introduced:

Audio
Video
Canvas
SVG
Local Storage
Session Storage
Semantic Elements
Form Validation
Drag and Drop
Geolocation
46. What is Geolocation API?

Answer:

It allows websites to access the user's geographical location.

47. What is Drag and Drop in HTML5?

Answer:

It allows users to drag an element and drop it somewhere else.

48. Which HTML tag is used for inserting images?

Answer:

<img>
49. Which HTML tag is used for creating hyperlinks?

Answer:

<a>
50. What is HTML5 mainly used for?

Answer:

HTML5 is used to create the structure and content of modern web applications and websites. It is the foundation of Front-End Development and an essential technology for Java Full Stack Developers.

Summary

These HTML5 Interview Questions cover:

HTML Basics
Tags and Attributes
Lists and Tables
Forms and Validation
Audio and Video
Semantic HTML
HTML5 APIs
Local Storage
Canvas and SVG

These questions are very useful for Freshers, Java Full Stack Developer interviews, and Front-End Developer interviews.
